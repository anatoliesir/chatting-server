# Chat Server
A modern, real-time public chatting application built with a focus on clean code, scalability, and strict architectural separation.

## Tech Stack & Tools
* Backend: ASP.NET Core Web API (.NET 9)
* Frontend: Blazor WebAssembly / Server
* Database: PostgreSQL via Entity Framework Core (EF Core)
* Real-time Communication: SignalR Hubs
* Architectural Patterns: Clean Architecture & CQRS (Command Query Responsibility Segregation) via MediatR

> In the future, I will learn more and more useful tools, for example Docker Container, etc..., maybe even connect a desktop app (like WPF).

## How did I achieve this?
I used Google Gemini to learn every tool, step by step, creating more than one project so I can memorize it well.\
